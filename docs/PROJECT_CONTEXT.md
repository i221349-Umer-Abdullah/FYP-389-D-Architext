## 1. Project Overview

ArchiText is a **Text-to-BIM** system: the user types a natural language description of a house (e.g. *"3 bedroom house with 2 bathrooms, kitchen, and living room"*), and the system generates:

1. A **2D floor plan** layout (room positions, sizes, adjacency)
2. A **3D IFC model** file (importable into Revit, BIMVision, etc.)

The AI pipeline is:
```
User Prompt → NLP (T5) → Room Spec → GNN (StructuralGNN) → Room Graph → IFC Generator → .ifc file
```

## 4. How to Run

### Step 1: Preprocess the ResPlan Dataset
Open Jupyter and run all cells in `notebooks/pre-processing.ipynb`:
```powershell
.\venv\Scripts\jupyter.exe notebook
```
This reads `ResPlan.pkl` (17,000 plans) and produces graph data in `data/processed/`.

### Step 2: Train the GNN
Run all cells in `notebooks/gnn-phase.ipynb`.
- Trains a **StructuralGNN** (attention-based, deterministic, no latent space)
- Saves checkpoints to `models/resplan_gnn/` every 10 epochs
- Best model saved as `gnn_best.pt`
- Training takes ~45–60 mins on an RTX 3080 with the recommended parameters

### Step 3: Run the Pipeline
```powershell
.\venv\Scripts\python.exe scripts\test_pipeline.py
```
Edit line 12 of `test_pipeline.py` to change the input prompt. Outputs:
- **2D preview:** `output/api_generated/2d_preview.png`
- **3D IFC file:** `output/api_generated/<job_id>.ifc`

### Step 4: View the 2D Layout
```powershell
.\venv\Scripts\python.exe scripts\visualize_2d.py
```
Generates a labeled 2D floor plan PNG from the last pipeline run.

## 6. Model Details

### StructuralGNN (current production model)
- **Architecture:** Attention-based GNN (4 layers, 256 hidden dim, 4 heads)
- **Type:** Deterministic (not a VAE/CVAE — no latent space)
- **Input:** 18-dim condition vector (room counts + unit type)
- **Output per node:** 16-dim feature vector = `[room_type×13 (one-hot), area_norm, cx_norm, cy_norm]`
- **Room types (13):** wall, bedroom, bathroom, living, kitchen, balcony, storage, parking, garden, pool, stair, veranda, inner
- **Max nodes:** 20
- **Trained on:** 17,000 ResPlan floor plans (57,800 after 4× flip augmentation)
- **Best checkpoint:** `models/resplan_gnn/gnn_best.pt`

### Condition Vector Format (18-dim)
| Index | Field |
|---|---|
| [0:9] | Room counts / 5.0 (bedroom, bathroom, kitchen, living_room, dining_room, balcony, garden, storage, parking) |
| [9] | total_rooms / 16 |
| [10] | is_apartment |
| [11] | is_house |
| [12:18] | padding zeros |

| File | Description |
|---|---|
